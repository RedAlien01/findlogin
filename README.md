# [ Installation ]
```
$ apt update upgrade
$ apt install python3 git
$ git clone https://github.com/penucuriCode/findlogin
$ cd findlogin
$ pip install -r requirement.txt
$ pip install asyncio aiohttp time socket request
$ python findlogin.py
```
